================================================================================
IMAGE SPECIFICATIONS
================================================================================

DEVICE IMAGES:
  Location: /assets/images/devices/{device-slug}.jpg
  Size: 500x400px (landscape)
  Format: JPEG
  Max: 200KB per image
  Fallback: /assets/images/devices/placeholder.jpg

SD CARD IMAGES:
  Location: /assets/images/sdcards/{card-id}.jpg
  Size: 200x140px (portrait/tall)
  Format: JPEG
  Max: 100KB per image
  Fallback: /assets/images/sdcards/placeholder.jpg

================================================================================
TEMPLATE VARIABLES AVAILABLE
================================================================================

In device page template (src/templates/device.html):
  {{DEVICE_IMAGE}} - Featured device image URL

The generator automatically fills in:
  - sdcard images from sdcardsMap[cardId].imageUrl
  - Device images from device.imageUrl
  - Falls back to placeholders if missing
